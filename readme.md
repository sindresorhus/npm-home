# npm-home [![Build Status](https://travis-ci.org/sindresorhus/npm-home.svg?branch=master)](https://travis-ci.org/sindresorhus/npm-home)

> Open the npm page of a package


## Install

```
$ npm install --global npm-home
```


## Usage

```
$ npm-home --help

  Usage
    $ npm-home [name]
    $ nh [name]

  Examples
    $ npm-home
    $ npm-home chalk
```
