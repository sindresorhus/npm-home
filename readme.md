# npm-home

> Open the npm page, Yarn page, or GitHub repo of a package

## Install

```sh
npm install --global npm-home
```

## Usage

```sh
$ npm-home --help

  Usage
    $ npm-home [name …]
    $ nh [name …]

  Options
    --github -g  Open the GitHub repo of the package
    --yarn   -y  Open the Yarn homepage of the package

  Examples
    $ npm-home
    $ npm-home chalk -g
    $ npm-home execa ava -y
```
